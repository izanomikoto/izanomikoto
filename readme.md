##  Projetos em Destaque

###  [Mundo RPG](https://github.com/izanomikoto/WorldRPG)
**Bot Discord RPG Completo** - Sistema de RPG textual com mais de 40 zonas, combate tático, economia viva e progressão vertical/horizontal.

**Tecnologias:** TypeScript, Discord.js v14, MongoDB, Node.js 20  
**Destaques:**
-  Sistema de combate tático turn-based
-  40+ zonas detalhadas com exploração rica
-  Economia completa (lojas, mercado, leilões)
-  Sistema de crafting avançado
-  Facções, guildas e sistema social
-  Geração procedural de conteúdo
-  Integração com IA (Google Gemini/OpenAI)

**Status:**  Production Ready |  100% Mundo Mortal Completo

##  Filosofia de Desenvolvimento

> "Código limpo não é escrito seguindo regras. Você não se torna um artesão de software aprendendo uma lista do que fazer e não fazer. Profissionalismo e artesanato vêm de valores que dirigem disciplinas."
> 
>  Robert C. Martin (Uncle Bob)

### Meus Princípios:

-  **Código Limpo** - Legibilidade e manutenibilidade em primeiro lugar
-  **Testes** - Confiança através de testes automatizados
-  **Documentação** - Código que se explica + documentação clara
-  **Refatoração** - Melhoria contínua do código
-  **SOLID** - Princípios de design orientado a objetos
-  **Performance** - Otimização sem sacrificar legibilidade
